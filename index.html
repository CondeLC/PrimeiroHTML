Meta Tags.

As meta tags são vistas apenas pelo navegador onde vc pode, dar um título ao seu site, vc pode adicionar uma descrição, vc pode dar detalhes para q funcione em qualqueer navegador, palavras chaves também, você pode identificar o autor da pagina, configurações para que a pagina se adeque até mesmo em no internet explorer, inserimos também o idioma para que apareça a opção de traduzia a página

Formatação.

  • p> significa paragrafo, tudo q ficar dentro dessa tag será um paragrafo
  • b> NEGRITOpalavras ou frases dentro dessa tag ficaram em negrito
  • i> itálico
  • u> sublinhado
  • strong palavra a ser realçada no texto como uma paravra chave, uma paravra com ênfase, diferente do negrito q muda somente a cor
  • hr> faz uma quebra de linha com risco de separação
  • del> frase cortada como se estivesse um valor antes e agora uma promoção
  • (< a href=" link da internet desejado ou #ID refentente ") é usado para ancorar, linkar palavras para uma navegação mais dinamica.
  • hr> risco de separação

Elementos de citação.

Quando dizemos coisas abrevidas em textos podemos usar usa uma explicação de breviatura por exemplo html ou WWW

Usamos "adress>" pare representar endereços de rua, localização ou endereço de email ou algum site também

lucas.conde.mattos@gmail.com

podemos usar cite> pare citações de obras, as letras ficam como itálico "No filme Titanic"

usamos também q> para quando citamos uma frase de um filme, formatando entre aspas a frase Rose vai para o lado que cabe nois ai facinho frase épica do filme TITANIC

Quando queremos dar distaque a uma frase que sera dita em seu texto para chamar atenção do leitor usamos

Rose vai para o lado que cabe nois ai facinho
Nesse trecho lembraremos de uma frase épica do filme Titanic onde Jack representado por Leonardo D'icaprio disse desesperadamente Rose vai para o lado que cabe nois ai facinho
